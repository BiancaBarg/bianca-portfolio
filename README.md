# Bianca Bargan — Portfolio

A responsive, accessible one-page portfolio for a 3D character animator, built with plain HTML, CSS, and JavaScript.

## Preview locally

```bash
python3 -m http.server 4173 --directory .
```

Then open `http://localhost:4173`.

## Content notes

- The “Old Reel” embeds the existing Vimeo reel from biancabargan.com.
- The “New Reel” is clearly marked as coming soon.
- Instagram, LinkedIn, and email use the contact details from the current portfolio.
- Replace copy or add project case studies directly in `index.html` when new material is ready.
